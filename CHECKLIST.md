# AI 입학사정관 — 준비사항 체크리스트

## 즉시 (Phase 0 전)

- [ ] **개발 환경 세팅**
  - [ ] Node.js 18+ 설치
  - [ ] Python 3.11+ 설치
  - [ ] Docker Desktop 설치 (PostgreSQL + Redis 로컬 구동용)
  - [ ] Git 레포 생성

---

## 2주 내 (Phase 1 시작 전)

- [ ] **data.go.kr API 키 발급**
  - https://www.data.go.kr 회원가입
  - "교육부 대학정보공시" API 활용 신청
  - 승인까지 1~3일 소요
  - 발급된 API 키를 `.env`에 설정

- [ ] **어디가(adiga.kr) 데이터 접근 방식 결정**
  - 선택지 A: 웹 스크래핑 (법적 리스크 있음)
  - 선택지 B: CSV 수동 다운로드 (느리지만 안전)
  - 선택지 C: 한국대학교육협의회에 데이터 파트너십 문의
  - → 법률 자문 받거나, 협의회에 먼저 문의 권장

- [ ] **Claude API 키 확보**
  - https://console.anthropic.com 가입
  - API 키 발급 + 결제 수단 등록
  - 예상 비용: 세특 1건 ~$0.03, 학급 32명 기준 월 ~$30~50

---

## 6주 내 (Phase 2 시작 전)

- [ ] **호스팅/배포 환경 결정**
  - 프론트엔드: Vercel (무료 티어 가능) or AWS Amplify
  - 백엔드: AWS EC2/ECS or GCP Cloud Run or Railway
  - DB: AWS RDS PostgreSQL or Supabase
  - 예상 월 비용: $50~200 (초기 규모)

---

## 장기 (Phase 4 이전)

- [ ] **개인정보보호 법률 검토**
  - 미성년자(고등학생) 성적 데이터 → 개인정보보호법
  - 학교/교육청 데이터 처리 위탁 계약 필요 여부
  - 개인정보처리방침 작성

- [ ] **파일럿 학교 섭외**
  - 실제 교사 1~2명에게 프로토타입 시연
  - 피드백: UI, 세특 AI 품질, 합격 예측 신뢰도
  - 실제 입결 데이터로 예측 정확도 검증

- [ ] **상업 데이터 파트너십 탐색** (선택)
  - 진학사(jinhak.com): 수험생 합불 결과 (~20만명/년)
  - 없어도 MVP(통계 기반)는 가능
